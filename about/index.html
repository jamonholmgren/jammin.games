Hi! I'm Jamon Holmgren, a software developer, entrepreneur, and indie game developer based in the beautiful Pacific Northwest.

Background

I started programming at age 12 on a 286, writing games in QBasic.

I never really stopped coding, and eventually co-founded Infinite Red, a mobile app consulting company.

Game Development Journey

While I've been a professional web and mobile developer for over two decades, game development was where I started way back in the early 90s, building games using QBasic. I eventually graduated to Gamemaker and web games, but always as small fun projects, never anything serious. In 2024, inspired by my brother Denton, I decided to dive deep into Godot development and build the game I've been wanting to make my whole life: a combat helicopter game like my favorite game of all time, Gunship 2000.

Beyond Code

When I'm not coding or building games, you might find me playing rec league ice hockey as a goalie, puttering around my 3 acre property on my Kubota tractor, and spending time with my family (including my new granddaughter).
