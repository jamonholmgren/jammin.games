Hi! I'm Jamon Holmgren, a software developer, entrepreneur, and indie game developer based in the beautiful Pacific Northwest.

Background

I started programming at age 12 on a 286, writing games in QBasic. That early exposure to coding sparked a lifelong passion for software development. Today, I'm the co-founder and CTO of Infinite Red, a React Native consulting company, and in my spare time, I build games using the Godot game engine.

Game Development Journey

While I've been a professional web and mobile developer for over two decades, game development was where I started and where my interests have remained. I built games the whole time using QBasic, Gamemaker, and the web, but never launched any of them. But in 2024, inspired by my brother Denton, I decided to dive deep into Godot development.

This website is where I'll be sharing my game development projects going forward!
